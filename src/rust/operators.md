# Operators

## Arithmetic

`+, -, *, /, %`

> Also + is used for array and string concatenation.

## Comparison

`==, !=, <, >, <=, >=`

## Logical

`!, &&, ||`

## Bitwise

`&, |, ^, <<, >>`

## Type casting

`as`

```rs
let a = 15;
let b = (a as f64) / 2.0; // 7.5
```

## Borrowing

`& or &mut`

## Dereference

`*`
